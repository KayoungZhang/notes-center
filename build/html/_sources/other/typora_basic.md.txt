## Typora基本教程



### 简介

Typora是一款轻便简洁的Markdown编辑器，支持即时渲染技术，这也是与其他Markdown编辑器最显著的区别。

### typora语法

typora软件大部分语法都可以通过快捷键和鼠标右键来完成，具体如下：



#### 标题级别

使用简单的 `ctrl+数字键` 就可以快速完成各种级别的标题



#### 高亮显示

`==文字==` 文字即高亮

==高亮显示==



#### 下划线

ctrl + u

<u>下划线</u>



#### 斜体

使用 `*单个星号*` 或者 `_单下划线_` 可以字体倾斜。快捷键 `Ctrl + I`

*斜体*



#### 加粗

使用 `**两个星号**` 或者 `__两个下划线__` 可以字体加粗。快捷键 `Ctrl + B`

**加粗**



#### 删除线

使用`~~删除线~~` 

~~删除线~~



#### 引用文字

\> + 空格 + 引用文字

> 这是一段引用文字
>
> >  换行



#### 无序列表

\-或\+或\* + space + enter

- AAA

- BBB

- CCC

  

#### 嵌套无序列表

在删除第一级后，自动跳到下一级，返回上一级，连续按次enter



#### 有序列表

数字键 + space + enter

1. AAA

2. BBB

3. CCC

   

#### 任务列表

鼠标右键选择对勾即可。



#### 代码块

- 插入单句或单词，使用反引号（`） 把其包围

  `prinf();`

- 插入代码段，输入3个反引号（`） + enter，并在后面选择一个语言名称即可实现语法高亮。

  ```python
  def helloworld():
      print("hello,world!")
  ```



#### 表格

使用鼠标右键插入表格或快捷键Ctrl + T。



#### 分割线

输入 `***` 或者 `---` 再按回车即可绘制一条水平线，如下：

-------



#### 脚注

`文字[^脚注]`

这是一个脚注[^1]



#### 目录（TOC）

输入 `[toc]` 然后回车，即可创建一个“目录”。TOC从文档中提取所有标题，其内容将自动更新。



#### 插入图片

直接拖进来或鼠标右键插入图像



#### 表情符号

英文状态下的引号，输入字母自动显示表情符号



#### 链接

`[链接名](http://example.com/ "标题")`

[链接](http://example.com/ "标题")

`[链接名](http://example.net/)`

[链接](http://example.net/)
